Firstname and Lastname:

1. First, you have to put a tag in the body which is the script tag.



2. Next, you have to create a variable. With the variable, you have to type first name.



3. Then, you have to type a name inside the quotes, and also have to create another variable with lastname.



4. Also, if you want to connect the firstname and the lastname you have to use variable fullname with concat().



5. Finally, you need to pring the fullname by console.log.





Please right click on your mouse and click inspect. Then go to console to see the example.
